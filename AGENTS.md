# Review Guidelines for Codex

## Code Review
- Check for security vulnerabilities (XSS, injection, etc.)
- Verify all frontmatter fields are correct and complete
- Ensure no hardcoded secrets or API keys
- Check for broken imports or missing dependencies
- Verify build passes after changes

## Content Review
- Fact-check pricing claims (AI tools, subscriptions)
- Flag outdated information about AI models or features
- Check that sources/URLs are real and not fabricated
- Ensure no first-person income claims in Money category articles ("I made $X")
- Money articles should use verified third-party case studies only
- Verify article excerpts are 140-180 characters

## Style
- EgoistAI voice: bold, direct, slightly irreverent
- No corporate fluff or generic AI filler phrases
- No "In today's rapidly evolving landscape..." type openings
